XML命名空间和XML Schema

2017-08-13

概念

XML 命名空间提供避免元素命名冲突的方法 。标签可以放入命名空间中，不同的命名空间中的相同名称标签是不同的标签。

命名冲突

在 XML 中，元素名称是由开发者定义的，当两个不同的文档使用相同的元素名时，就会发生命名冲突。
