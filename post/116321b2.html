环境变量与用户自定义变量的区别

环境变量是全局变量，用户自定义变量是局部变量。
用户自定义变量只在当前的 shell 中生效，环境变量在当前 shell 和这个 shell 的所有子 shell 中生效。
用户可以自定义环境变量，但对系统生效的环境变量名和变量作用是固定的。

常用的环境变量

HOSTNAME：主机名
SHELL：当前的shell
TERM：终端环境
HISTSIZE：历史命令条数
SSH_CLIENT：当前操作环境是用 ssh 连接的，这里记录客户端 ip
SSH_TTY：ssh 连接的终端是 pts/1
USER：当前登录的用户
