Linux文件管理02 文件的查看和处理

发表于 2018-05-05

目录

主目录，家目录：HOME

ls 显示指定路径下的文件列表

tree 显示目录树

pwd 显示当前目录（print working directory）

cd 切换目录（change directory）

mkdir 建立目录（make directories）

rmdir 删除空目录（remove empty directories）

文件

file 查看文件类型

查看文件是文本还是二进制等

stat 查看文件的元数据

查看Linux 时间戳管理

rm 删除文件或目录（remove）

cp 复制命令（copy）

源是文件结果一定是有一个文件产生，源是目录结果一定是有一个目录产生

目标
目标文件不存在目标目录不存在文件目录
不存在同名存在同名
源文件创建目标文件，将源文件内容复制到目标文件报错，先使用mkdir创建目标目录覆盖新建同名文件

覆盖
目录(-r)报错创建目标目录，将源目录中的文件复制到目标目录报错新建同名目录覆盖

mv 剪切或改名命令（move）
